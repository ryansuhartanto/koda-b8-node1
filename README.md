# `arrEvenOdd`

A program that filter an array of even or odd numbers.
Implemented in using both JavaScript built-in Array method (`index.js`) and
pure operators (`index-manual.js`).

## Usage

```js
const ARRAY = [1, 2, 3, 4];
console.log(arrEvenOdd(ARRAY, "even")); // returns [2, 4];
console.log(arrEvenOdd(ARRAY, "odd")); // returns [1, 3];
```
